"# Web sajt za porucivanje i placanje hrane putem Interneta, koristeci C#, ASP .NET MVC, Stripe API kao sistem placanja. MSSQL kao bazu i deployment na Heroku. Postoje uloge administratora i korisnika, kao i jos par uloga koje su vezane za interni rad sistema.  Autorizacija je omogucena pomocu Facebook-a ili Google-a, ali takode postoji i klasicna registracija putem sajta. Omogucene su CRUD operacije, kako za korisnika tako i za administratora i interne radnike."
